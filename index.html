About

Boxkid vs. the Ocean is a (yet to be released) webcomic about a character named Boxkid, who dreams of fighting the ocean. That's pretty much it.

Preview

Here are some drawings I've made related to Boxkid. These may contain spoilers so they will be hidden once the series actually begins. Note that this is already very similar to what actual panels will look like.

When will it release?

Boxkid will begin it's run in 2024. I'm currently writing the first chapter so I'm not entirely sure how long it will take. Each chapter will release when ready, and there is no set schedule. I am trying to find ways to notify people of new chapter releases, but checking this website once in a while is the best way of staying in the know for now.

Where can I read it?

This website will be the best place to read the comic once it is out. I will host two versions of the comic. One will be the orginal square page format that is the intended way to read it, but I will also host a scroll format version for easier reading on mobile devices. The following are other platforms that I will post the comic to, as well as which version they will feature:

Newgrounds: Page version
Webtoon: Scroll version
Tapas: Scroll version

Links to these will be provided once the comic launches.
