About

Boxkid vs. the Ocean is a webcomic that is currently in development by FLARE145. It will feature a character named Boxkid as he tries to do anything at all. The comic will hosted here in a 1:1 square page format, but it will likely be posted on webtoon as well in a scrolling format. This website is still a work in progress so don't be confused when something doesn't work or seems unfinished. I'm trying to stay really casaul about this project but I will still put plenty of effort into it, so I hope you enjoy.
