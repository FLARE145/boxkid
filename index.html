About

Boxkid vs. the Ocean is a (yet to be released) webcomic about a character named Boxkid, who dreams of fighting the ocean. That's pretty much it.

When will it release?

Boxkid will begin its run very soon. I'm currently writing the first chapter so I'm not entirely sure how long it will take. Each chapter will release when ready, and there is no set schedule. An rss feed will be available soon for anyone who would like to keep tabs on this project, and I am working on more ways to get notified of its release as well.
